# Testapp 16 #

## Instuctions ##

Clone project:
```
$ git clone git@gitlab.ida.liu.se:wilbe075/testapp16.git
```

Reset database:
```
$ rake db:schema:load
```

Seed database:
```
$ bundle exec rake db:seed
```

Run all RSpec tests:
```
$ rspec
```
